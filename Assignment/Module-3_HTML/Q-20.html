Q-20. What are logical and physical tags in HTML?

-> Logical Tags :

- These tags are also known as Semantic Elements.
- These tags are used to add some logical or semantic value to the text.
- Example :
<aside>, <header>, <footer>, <nav>, <section>

-> Physical Tags :

- These tags are also known as Structural Elements.
- These tags are used to provide the visual appearance to the text.
- Example :
<b>, <i>, <u>, <strong>, <sup>, <sub>
