Q-7. What is the difference between the ‘id’ attribute and the ‘class’ attribute of HTML elements? With Example.

-> ID :

- Id attribute is unique, so this is used only one time in the webpage.

- Example:

Welcome to My Website

This is some content.

-> Class :

- Class attribute is used multiple times in the webpage.

- Example:

This is an important paragraph.

This is another important paragraph.
