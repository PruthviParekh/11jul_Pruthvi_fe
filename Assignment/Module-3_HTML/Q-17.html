Q-17. What are the different tags to separate sections of text?

- <br> Tag : This tag is used to breaks the line.
- <p> Tag : This tag is used to write a paragraph of text.
- <blockquote> Tag : This tag is used to define large quoted sections.
- <div> Tag : This tag is used to group and separate content into distinct sections.
- <header> & <footer> Tag : These tags are used to define the header and footer sections of a document or a section within a document.
