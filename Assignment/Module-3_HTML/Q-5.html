Q-5. What are different types of lists in HTML? With Example.

- There are three types of lists.
  1>Ordered List
  2>Unordered List
  3>Description List

- 1>Ordered List :

   -  An Ordered List is used to create a list of items that are ordered or numbered in a sequential manner.

   - Example :

  1. Item 1
  2. Item 2
  3. Item 3
- 2>Unordered List :

   -  An Unordered List is used to create a list of items where the order of the items is not important.
   -  It is typically represented as a bulleted list, with each item in the list preceded by a bullet point or some other marker.

   - Example :
  • Item 1
  • Item 2
  • Item 3
- 3>Description List :

   -  A Description List is used to define a list of terms and their corresponding descriptions or definitions
