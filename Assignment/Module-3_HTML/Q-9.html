Q-9. How is Cell Padding different from Cell Spacing? With Example.

- Cell padding and cell spacing are two attributes and it is commonly used in HTML tables to control the spacing and alignment of table cells.

-> Cell Padding :

- It specifies the distance between the cell's content and its border.

- Example :

Cell 1 Cell 2

-> Cell Spacing :

- It controls the spacing between neighboring cells within the table.

- Example :

Cell 1 Cell 2
