Canvas and SVG tags

1> Canvas:-

-> Canvas is used to draw graphics.
-> It can be animated using JavaScript or CSS animations.
-> Canvas tag is only container of graphics and actual graphics is draw with JavaScript.
-> Canvas is raster base graphics.
-> Canvas is blure when user can zoom the canvas because it use pixels.
-> In HTML5, you can create a canvas element by adding a <canvas> element in your code.

Example:-


Code:-
<script>
var c = document.getElementById("myCanvas");
var ctx = c.getContext("2d");
ctx.fillStyle = "red";
ctx.fillRect(50, 50, 100, 100);
</script>

2> SVG:-

-> SVG stands for Scalable Vector Graphics.
-> SVG is vector based graphic.
-> SVG is scalable without losing quality.

Example:-


Code:-
<svg width="100" height="100">
<circle cx="50" cy="50" r="40"
stroke="red" stroke-width="4" fill="green" />
</svg>
