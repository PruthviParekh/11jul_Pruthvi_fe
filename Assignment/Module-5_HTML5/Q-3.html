Semantic element in HTML5?

- Sementic element have its used create for the structure, it's tags have given it's name according it's work.

Example :-

1> nav
2> header
3> footer
4> article
5> section
6> aside

1> nav :- Navigation used for create the menu or some other option.
Ex :- <nav></nav>

2> header :- It use to create header section in webpage it include web-logo
Ex :- <header></header>

3> footer :- It defines footer content that is after the main content.
Ex :- <footer></footer>

4> article :- It is used for some article,poster etc content.
Ex :- <article></article>

5> section :- It is used for saw all type content in web page.
Ex :- <section></section>

6> aside :- It used for side-content in webpages.
Ex :- <aside></aside>
