In Bootstrap 4, what is flexbox?

        - Flexbox is a layout model that allows you to design flexible and responsive layouts for your web pages.

        - Some classes of flexbox are given below :

        1> flex-row: create flex container with row layout
        2> flex-column: create flex container with column layout
        3> justify-content-start : align items in the main axis from start to end 
        4> justify-content-end : align items in the main axis from end to start
        5> justify-content-center : align items in the main axis from center
        6> align-items-start : align items in the cross axis from top to bottom
        7> align-items-end : align items in the cross axis from bottom to top
        8> align-items-center : align items in the cross axis from center
