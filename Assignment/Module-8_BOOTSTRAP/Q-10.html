What is a bootstrap card and how would you create one?

- A Bootstrap Card is a flexible container with header, body, and footer sections.
- It can contain various elements like images, headings, paragraphs, buttons, etc., making it highly customizable.
- To create card add 'class="card"' to the div that contains all three parts of the card:
- For Header use 'class="card-header"',
Example :- <div class="card-header></div>
- For Body use 'class="card-body"',
Example :- <div class="card-body></div>
- For Footer use 'class="card-footer"'. It is optional.
Example :- <div class="card-footer></div>
