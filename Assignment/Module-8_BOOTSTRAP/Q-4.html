What do you know about the Bootstrap Grid System?

- The Bootstrap Grid System is a powerful layout utility provided by Bootstrap.
- Bootstrap Grid System is a popular front-end framework for web development.
- It allows developers to create responsive and flexible layouts for their websites easily.
- The grid system is based on a 12 column layout. Each row can contain one or more columns.
- Bootstrap provides responsive classes to control column behavior at different screen sizes, xs, sm, md, lg.
- xs : Extra small devices (phones, less than 768px)
- sm : Small devices (tablets, between 768px and 992px)
- md : Medium devices (desktops, between 992px and 1200px)
- lg : Large devices (large desktops, greater than 1200px).
