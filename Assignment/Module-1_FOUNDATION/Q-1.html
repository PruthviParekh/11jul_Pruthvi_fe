-> Website :

- Website is a collection of multiple webpages.
- There is two types of Website.
  1>Static Website
  2>Dynamic Website

- 1>Static Website
  ->Static website means its data is not stored anywhere.
  ->Html, CSS and JavaScript are part of the Static Website.

- 2>Dynamic Website
  ->Front-end, Back-end and Databases are used in Dynamic Website.
  ->In the Dynamic Website, HTML is used to give the structure of the website, CSS is used to give styles in website, and JavaScript is used to to process without storing the data.
- Platforms of run the website,
  Chrome, Firefox, etc.

-> Http :

- Http stands for Hypertext Transfer Protocol.
- It is used to transporting informations between Client and a Web server.
- It is less secure than https.

-> URL :

- URL stands for Uniform Resource Locator.
- It is used to specify addresses on the World Wide Web.
