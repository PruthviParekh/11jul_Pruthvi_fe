Array vs object defences in JS?

1> Array :

- Ordered Collection : Arrays are ordered collections of elements. Each element in the array has a numeric index starting from 0.

- Access by Index : Elements in an array are accessed using their index. For example, array[0] retrieves the first element of the array.

- Methods and Properties : Arrays come with a variety of built-in methods for common operations like adding, removing, and iterating over
  elements (e.g., push(), pop(), shift(), unshift(), forEach(), map(), filter(), etc.).

- Use Cases : Arrays are typically used when the order of elements matters or when you need to perform operations on a list of items.

- Syntax : Arrays are defined using square brackets [].

- Example

  <script>
      let array = [1, 2, 3, 4];
      console.log(array)
  </script>

- Output :

  (4)Â [1, 2, 3, 4]



2> Objects :

- Unordered Collection : Objects are collections of key-value pairs where the keys (also known as properties) are strings (or Symbols),
  and the values can be any type.

- Unordered Collection : Objects are collections of key-value pairs where the keys (also known as properties) are strings (or Symbols),
  and the values can be any type.

- Methods and Properties : Objects do not have the array-specific methods but come with their own set of methods like Object.keys(),
  Object.values(), Object.entries(), and Object.assign() for working with object properties.

- Use Cases : Objects are used when you need to store data as key-value pairs, often to represent real-world entities or structured data
  where each key has a specific meaning.

- Syntax : Objects are defined using curly braces {}.

- Example :

  <script>
      let object = {
      key1: 'value1',
      key2: 'value2',
      key3: 'value3'
      };
      console.log(object)
  </script>

- Output : 

  {key1: 'value1', key2: 'value2', key3: 'value3'}
