How can the style/class of an element be changed?

- In HTML and CSS, you can change the style or class of an element using JavaScript.

- You can use the 'getElementById()' method to select the desired element by its ID.

- Example :-

    var headingElement = document.getElementById("myHeading");
    headingElement.style.color = "blue";
    headingElement.style.fontWeight = "normal";
    headingElement.style.border = "5px solid blue";
