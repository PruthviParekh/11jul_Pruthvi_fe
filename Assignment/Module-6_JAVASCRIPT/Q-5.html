What are undeclared and undefined variables?

->Undeclared Variables:-

- Undeclared variables are the ones that are used without explicit declaration using any of the keyword like var, let, const.
- If we use ‘typeof’ operator to get the value of an undeclared variable, we will face the runtime error with the return value as “undefined”.

- Example :-

console.log(x)

->Undefined Variables:-

- Undefined Variables indicates that a variable has not declared at all.

- Example :-

var x;
console.log(x); //Output : "undefined"
