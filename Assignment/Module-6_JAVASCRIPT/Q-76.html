What are all the looping structures in JavaScript?

- Loop is used to execute the line of statement multiple times.

- There are two types of loop.

- 1> Entry Control Loop :
    
  For loop
  For-in loop
  For-of loop
  For-each loop
  While loop
        
- 2> Exit Control Loop :
        
  Do-while loop
    
- Example :

1. For Loop

- Syntax :

        for(condition)
        {
            //code
        }

- Example :

        for(let i = 0; i <= 5; i++)
        {
            console.log("The value of i is : " + i);
        }
        

2. For-in Loop

- Syntax :

        for(condition)
        {
            //code
        }

- Example :

        let i;
        let array = [10, 20, 'Hello', 'World', '*'];
        for(i in array)
        {
            document.write(i + ',')
        }   
        

- 3. For-of Loop

- Syntax :

        for(condition)
        {
            //code
        }

- Example :

        let i;
        let array = [10, 20, 'Hello', 'World', '*'];
        for(i of array)
        {
            document.write(i + ',')
        }
        

- 4. While Loop

- Syntax :

        while(condition)
        {
            //code
        }

- Example :

        let i = 0;
        while(i < 5)
        { 
            console.log("Hello World!!!"); 
            i++;
        }
        

- 5. Do-while Loop

- Syntax :

        do
        {
            //code
        }while(condition);

- Example :

        let x = 10;
        do
        {
            console.log(x);
            x--;
        }while(x >= 0);
