What is the difference between && and || in JavaScript?

1> Logical AND ('&&') :

- Symbol : &&
- Operation : Returns true if both operands are truthy; otherwise, it returns false.
- Short-circuit evaluation : If the first operand is falsy, the second operand is not evaluated because the overall expression can never
  be true.
- Usage : Used to ensure multiple conditions are true.

- Syntax : condition1 && condition2

- Example : 

true && true   // true
true && false  // false
false && true  // false
false && false // false

let a = false && (console.log('This will not print'), true); //'This will not print' is not evaluated

2> Logical OR ('||') :

- Symbol : ||
- Operation : Returns true if at least one of the operands is truthy; otherwise, it returns false.
- Short-circuit evaluation : If the first operand is truthy, the second operand is not evaluated because the overall expression is
  already true.
- Usage : Used to ensure at least one condition is true.

- Syntax : condition1 || condition2

- Example :

true || true   // true
true || false  // true
false || true  // true
false || false // false

let b = true || (console.log('This will not print'), false); //'This will not print' is not evaluated

-> Combine Example :

<script>
    let x = 5;
    let y = 10;

    if (x > 3 && y < 15) {
        document.write("Both conditions are true"); //This will print
    }

    if (x > 6 || y < 15) {
        document.write("At least one condition is true"); //This will print
    }
</script>

- Output :

Both conditions are true
At least one condition is true
