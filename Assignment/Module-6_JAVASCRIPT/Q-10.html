What are all the looping structures in JavaScript?

- Loop is used to execute the line of statement multiple times.

- There are two types of loop.

- 1>Entry Control Loop

  • For loop
  • For-in loop
  • For-of loop
  • For-each loop
  • While loop
- 2>Exit Control Loop
  • Do-while loop

- Example:-

        1. For Loop
        for(condition)
        {
            //code
        }
        

        2. For-in Loop
        for(condition)
        {
            //code
        }
        

        3. For-of Loop
        for(condition)
        {
            //code
        }
        

        4. While Loop
        while(condition)
        {
            //code
        }
        

        5. Do-while Loop
        do
        {
            //code
        }while(condition);
