What is Condition Statement? 

1> if Statement :

- This statement is used to execute a block of code only if a specified condition is true. 

- Example :

if(condition)
{
    //code to be executed if the conditon is true
}
    
2> if...else Statement :

- This statement is used to execute a block of code only if a condition is true, and another block of code if the condition is false.

- Example :
        
if(condition)
{
    //code to be executed if the conditon is true
}
else
{
    //code to be executed if the condition is false
}
    
3> else-if ladder Statement :

- This statement allows you to check multiple conditions.

- It is used with conjunction with 'if' and 'else'.
        
- Example :
                
if (condition1)
{
    // Code to execute if condition1 is true
}
else if (condition2)
{
    // Code to execute if condition2 is true
}
else
{
    // Code to execute if both condition1 and condition2 are false
}  
        
4> switch case Statement :

- This statement is used to perform different actions based on different conditions.
            
- Example :
                    
switch(var)
{
    case value1:
        //code to be executed if var equals to value1
        break;
    case value2:
        //code to be executed if var equals to value1
        break;
    case value3:
        //code to be executed if var equals to value1
        break; 
    default:
        // Code to execute if none of the cases are true
}
