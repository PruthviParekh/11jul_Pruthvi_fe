What are the disadvantages of using innerHTML in JavaScript?

- When we write any text in innerHTML code, the old text will be removed automatically.

- Example :

  <div id="d1">
      <p>This is old content.</p>
  </div>
        
  <script>
      document.getElementById('d1').innerHTML+='<p>'This is new content';
  </script>

- Output :

    

        This is old content.
