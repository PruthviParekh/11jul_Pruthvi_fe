What is the use of isNaN function?

- The isNaN() function in JavaScript is used to determine whether a given variable or value is NaN (Not a Number).

- In JavaScript NaN means Not a Number.

- The 'isNaN' is a special value in JavaScript.

- 'isNaN' function return boolean value as output. 

- Example :-

    <script>
        var x = "Hello";
        if(isNaN(x))
        {
            document.write("It is not a number");
        }
        else
        {
            document.write("It is a number");
        }
    </script>
