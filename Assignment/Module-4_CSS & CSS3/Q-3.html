Q-3. What is the difference between CSS2 and CSS3?

- CSS3 version supports more browsers than CSS2.
- CSS3 version introduces some new selectors like, the attribute selector, the nth-child selector, and the adjacent sibling selector, that allows you to select specific elements.
- CSS3 version introduces properties like backgrounds, borders, animatons, transitions etc.
- CSS2 version introduces properties like, padding, margins etc.
