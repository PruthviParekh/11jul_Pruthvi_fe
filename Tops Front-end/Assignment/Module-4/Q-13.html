Q-13. What are the ways to integrate CSS as a web page?

- There are three ways to integrate CSS as a web page.
1>Inline CSS
2>Internal CSS
3>External CSS

1>Inline CSS :

- This type of CSS is apply using style attribute in any tag.
- Ex : <p style="color: red;">Lorem ipsum dolor sit amet.</p>

2>Internal CSS :

- This type of CSS is apply using style tag in head.
- Ex :
<head>
   <style>
           color: red;
   </style>
</head>

3>External CSS :

- For apply this type of CSS, create the css file using .css extension and appply it in the html file using href attribute in link tag.
- Ex :
<head>
   <link rel="stylesheet" href="css/style.css">
</head>

<body>
   <p>Lorem ipsum</p>
</body>
