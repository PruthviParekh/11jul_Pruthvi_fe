Q-19. What is difference between HTML and XHTML?

-> HTML :

- HTML stands for HyperText Markup Language.
- HTML is SGML based Language.
- HTML is used to create web and Android applications.

-> XHTML :

- XHTML stands for Extensible HyperText Markup Language.
- XHTML is an XML-based language.
- XHTML is also used to create web and Android applications.
