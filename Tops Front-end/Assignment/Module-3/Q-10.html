Q-10. How can we club two or more rows or columns into a single row or column in an HTML table? With Example.

- In HTML tables, you can merge two or more rows or columns into a single row or column using the rowspan and colspan attributes.

-> RowSpan :

- Example :

No Name City
1 Viraj Mumbai
2 Siraj
3 Miraj Chennai

-> ColSpan :

- Example :

Name Details City
Shyam Age: 25 Gender: Male Delhi
Sita Age: 28 Gender: Female Punjab
