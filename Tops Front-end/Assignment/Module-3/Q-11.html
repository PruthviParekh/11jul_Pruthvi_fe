Q-11. What is the difference between a block-level element and an inline element?

-> Inline-level Element :

- Inline elements never start from a new line.
- Height and Width of Inline Element is same as Content.

-> Block-level Element :

- Block elements start from the new line.
- The Height of the Block-level Element is same as Content.
- The Width of the Block-level Element is same as the screen size .
