Q-1. Are the HTML tags and elements the same thing?

- No, The Html tags and elements are not the same thing.
- Because, HTML Tags are building blocks of HTML Page and HTML Elements are components that are used in HTML Page.
- When the Start tag, Content, and End tag are combined, then it is called an Element.
- The size of an Element is same as the Content. - There is two types of Elements.
  1>Inline Level Element
  2>Block Level Element
