Q-4. What are HTML Entities? With Example.

- HTML Entities are special codes of characters.
- It is used to represent characters that are not easily represented directly in the code.

- Examples :

  1> < - represents Less than.
  2> > - represents grater than.
  3> " - represents double quotation mark.
  4> ' - represents single quotation mark.
  5> © - represents copyright.
