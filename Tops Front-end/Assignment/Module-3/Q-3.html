Q-3. What are void elements in HTML? With Example.

- A void element is an element whose content model never allows it to have contents under any circumstances.

- Examples :

 1> <img> : Used to give images.
         -> Ex : <img src="#" alt="image">
 2> <br> : Used to break the lines.
         -> Ex : <p>This is some text.<br>Here's a line break.</p>
 3> <input> : Used to give input fields in forms.
         -> Ex : <input type="text" name="username" placeholder="Enter your username">
 4> <link> : Used to link external resources like stylesheets.
         -> Ex : <link rel="stylesheet" href="styles.css">
