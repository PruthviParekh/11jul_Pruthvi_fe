Q-2. What are tags and attributes in HTML?

- 1>Tags :

   -  HTML tags are used to hold the HTML element.
   -  There is two types of Tags :
      1>Pair Tag
      2>Non-Pair Tag

     1>Pair Tag :

      - Pair tags require an opening tag that turns a formatting feature on and a closing tag that turns the feature off.
      - Example of Pair Tag : <p></p>, <title></title>, <div></div>, <script></script>, etc.

     2>Non-Pair Tag :

      - Non-Pair tags are single tags with no closing tag.
      - These tags are also called Singular Tags.
      - Example of Non-Pair Tag : br, hr, link, meta, etc.

- 2>Attributes :

- Attributes means Additional Information that tag.
- Attributes are used to describe the characteristic of an HTML element in detail.
- Example : Common attributes include "id", "class", "src", "href", "alt", etc.
